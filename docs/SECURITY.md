# Security Strategy

- Tenant isolation through workspace-scoped access.
- PostgreSQL row-level security for enterprise deployments.
- Signed URLs for uploads and downloads.
- File scanning before processing uploads.
- Prompt injection protection around retrieved content.
- Strict tool allowlists for AI agents.
- Usage limits by user, workspace, IP, and plan.
- Audit logs for exports, sharing, billing, and admin actions.
- Encryption in transit and at rest.
