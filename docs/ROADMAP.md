# MVP Roadmap

## Phase 1: Prompt to PPT

- Prompt studio
- Intent detection
- Presentation IR
- Academic/business templates
- PPTX export contract
- PDF export placeholder
- Auth integration points
- Usage metering

## Phase 2: Documents and research

- DOCX/PDF rendering
- Citation manager
- RAG over uploaded PDFs
- Charts and Mermaid diagrams
- Section-level regeneration

## Phase 3: Collaboration and design

- Live editing with Yjs
- Comments
- Version history
- AI design engine
- Voice input

## Phase 4: Enterprise

- SSO
- Team workspaces
- Admin controls
- Institution templates
- API access
- Private deployments
